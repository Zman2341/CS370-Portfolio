# CS370-Portfolio

#Project Overview
#This project focused on building an intelligent agent that represents a pirate navigating an 8×8 maze to find a treasure using deep Q‑learning. I was given starter code that included the maze environment (TreasureMaze), the experience replay class (GameExperience), and helper functions for observing the environment, taking actions, and checking completion. I created the full Q‑training loop myself, including the exploration/exploitation logic, storing episodes in memory, training the neural network, updating the target network, and running the completion check. I also wrote the code that controls how the agent learns from rewards and improves its pathfinding over time.

#Connecting This Project to Computer Science
#What computer scientists do and why it matters
Computer scientists solve problems by designing systems that can process information, learn patterns, and make decisions. This matters because almost every field—healthcare, transportation, finance, education, entertainment, relies on software and intelligent systems to work efficiently. Projects like this one show how AI can learn from experience and adapt to new situations, which is a major part of modern computer science.

#How I approach a problem as a computer scientist
#This course helped me think about problems in a more structured way. Instead of jumping straight to coding, I learned to break the problem into smaller pieces: understanding the environment, defining the agent’s actions, deciding how it learns, and testing whether it performs correctly. Reinforcement learning especially taught me to think in terms of states, rewards, and long‑term outcomes, which is a useful mindset for many real‑world problems.

#My ethical responsibilities
#Working with AI also comes with ethical responsibilities. As a computer scientist, I need to think about how the system affects the end user and the organization. That includes being careful with data, avoiding biased or harmful behavior, and making sure the system is transparent and safe to use. Even in a simple project like this, it’s important to understand how an autonomous agent makes decisions and how to prevent unintended consequences.
